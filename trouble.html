The Trouble Game

JavaScript Challenge

Foundation

Algorithm and JavaScript

When we learn programming, it will be impossible to master JavaScript without knowing algorithm, which means planning ahead in a simple term. And our task is to understand the rules of the Trouble game and code a HTML version of this game with JavaScript.

Languages

HTML CSS JavaScript

JavaScript takes about 90% of code in this project, more accurately, 1600 lines of JavaScript are working to made the game fully functional. It takes care about transfer data between pages, pegs spawning, moving, hitting, and winning the game.

Development

Tons of bugs

The mission won't be accomplished without running into couple of bugs, such as pegs randomly fly to a position that they shouldn't be, running into another player's FINISH line, and also not hitting opponent back to START. Luckly most of the bugs are syntax errors, so it is easy to fix.
